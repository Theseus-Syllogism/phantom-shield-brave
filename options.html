Phantom Shield

Bridge setup

Phantom Shield needs a local TLS-impersonation bridge. The steps below complete in any order.

  1. Install the bridge
    A small Python process (mitmproxy + curl_cffi + our addon) that runs in the background.
  2. Bridge addon up to date
    The mitmproxy addon must be recent enough to answer the probe.
  3. Configure Brave's proxy
    Routes Brave's HTTPS traffic through the local bridge.
  4. Trust the mitmproxy CA
    Required so the bridge can terminate + re-issue HTTPS connections.
